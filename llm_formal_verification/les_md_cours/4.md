```markdown
# Cours 5 : Application du prouveur KeY et états
**Auteur :** A. Giorgetti
**Pages / Diapositives :** 161 à 164

## Règle Loop Invariant dans KeY
1. **Chargement :** Charger le fichier `.key` et sélectionner la formule de boucle.
2. **Lancement :** Cliquer sur le bouton `Loop Invariant`.
3. **Paramétrage :** Attention avec KeY 2.12.3, le champ `Modifiable` doit être laissé par défaut (`allLocs`). Toute modification manuelle désactive les boutons `Apply` et `Store`.

## Application des Substitutions (Explicit State Update)
Une fois qu'aucune règle de simplification syntaxique ne s'applique, le séquent est sous la forme `C ⇒ {U} F, D` :
* `{U}` est une séquence de mises à jour (*updates*).
* `U` est traité comme une substitution explicite.
* **Réduction :** On applique `U` à `F` par application directe (valeur des variables dans l'état). 
* Si `{U, s} F`, on applique `s` à `F` d'abord, puis `U` au résultat.
* Distinguer les *sequential updates* des *parallel updates* dans la documentation officielle de KeY.
```

### Fichier 20 : Cours 5 - Exemples d'application de règles (Tant-que)
```markdown
# Cours 5 : Exemple d'application (Boucle)
**Auteur :** A. Giorgetti
**Pages / Diapositives :** 165 à 166

## Exemple : `while (k < n)`
Invariant de boucle `I` : `k = i ∧ k ≤ n`
Programme `p` : `{i = i + 1; k = k + 1;}`
Variables modifiées `V` : `i := i1, k := k1`

1. **Condition initiale :** `i = 0 ⇒ {k := 0} I`
2. **Préservation :** `i = 0 ⇒ {k := 0, i := i1, k := k1} ((I ∧ k < n) → [p] I)`
3. **Sortie :** `i = 0 ⇒ {k := 0, i := i1, k := k1} ((I ∧ ¬(k < n)) → (i = n))`

* Une fois la 3ème condition réduite par arithmétique et substitution, on conclut par transitivité et symétrie de l'égalité.
```

### Fichier 21 : Cours 4 - Règles de clôture dans KeY
```markdown
# Cours 4 : Clôture de sous-buts (Fol / Sequent Calculus)
**Auteur :** A. Giorgetti
**Pages / Diapositives :** 142 à 144

## Conditions de clôture (Proof Branch Closed)
Une branche de preuve peut être fermée si :
1. **Séquent valide :** 
   - `Γ, φ ⇒ φ, Δ` (close)
   - `Γ ⇒ true, Δ` (true)
   - `Γ, false ⇒ Δ` (false)
2. **Équation évidente :** `Γ ⇒ t = t, Δ` (eqClose)

## Fonctionnalités du prouveur KeY
* **Gestion multi-preuves :** *Task list*.
* **Navigation :** Arbres de preuve interactifs (Java Swing).
* **Technique :** Drag-and-drop pour l'instanciation de quantificateurs.
* **Automatisation :** Possibilité d'utiliser des solveurs externes (SMT solvers).
* **Manipulation :** Prise en charge des règles par menu contextuel (point-and-click).
```
